# AI-Powered-Weather-Application-with-Live-Chatbot

📌 About The Project

This is a beginner-friendly weather web application that shows real-time weather information of different cities using a weather API.
The project also includes an AI-powered live chatbot that helps users by answering weather-related questions.

I built this project to improve my skills in web development, API integration, and AI features.

🚀 Features
🌍 Search weather by city name
🌡️ Shows temperature and weather condition
💨 Displays humidity and wind speed
🤖 AI-powered chatbot support
📱 Responsive user interface
⚡ Real-time weather updates
🛠️ Technologies Used
HTML
CSS
JavaScript
React.js 
Node.js 
OpenWeather API
Gemini API / OpenAI API
